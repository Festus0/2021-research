# 2021 Research – ImageJ LoG/DoG & Cell Detection Toolkit

A cleaned and anonymised 2021 MSc research toolkit for automated image analysis using ImageJ macros.

## Key Capabilities
- Cell counting automation
- Foci detection workflows
- Laplacian of Gaussian (LoG) filtering
- Difference of Gaussians (DoG) filtering
- Coordinate extraction from microscopy images
- Reproducible image processing pipelines

## Repository Structure
macros/     - Core ImageJ macro scripts (.ijm)
templates/  - Template ImageJ macros
images/     - Example microscopy images and screenshots
latex/      - Supporting equations and LaTeX files
docs/       - Reports and summaries
